# 环境安装

使用Anaconda的安装虚拟pyqt5环境

### 0 环境

```bash
python=Python 3.8.13
pyqt5==5.12.3

```

### 1 问题总结

**Q1：使用默认版本pyqt5会出现安装错误，spyder等库和pyqt5发生冲突**

**Solution：**

需要下载的内容：[https://pypi.tuna.tsinghua.edu.cn/simple/pyqt5-tools/](https://pypi.tuna.tsinghua.edu.cn/simple/pyqt5-tools/ "https://pypi.tuna.tsinghua.edu.cn/simple/pyqt5-tools/")在网站中找到pyqt-tools5.12版本。

```bash
pip install pyqt5==5.12.3
pip install pyqtwebengine==5.12.1
pip install typed-ast
pip install wheel
pip install pyqt5_tools-5.12.3.1.6rc1-cp39-none-win_amd64.whl
```

如果pip check后发现还是有pyqt5-plugins不兼容的问题，卸载它（没有就不用了）
